Psychological Horror Mobile Game

Holly Bell is an atmospheric psychological horror game developed for mobile platforms (iOS/Android). As the Gameplay Programmer at Hesperian Studios, I was responsible for core systems development, optimization, and delivering a smooth 60 FPS experience on mobile devices.

The game combines storytelling with puzzle-solving elements in a dark, mysterious atmosphere. It was successfully selected for the 9 Eylül University Game Development Pre-Incubation Program and won multiple game development competition awards.

Key Features:

  • Psychological horror narrative
  • Puzzle-based gameplay mechanics
  • Atmospheric environments and lighting
  • Optimized for mobile 60 FPS performance
  • Complete Save/Load system
  • 12 cinematic sequences

My Responsibilities:

  • Designed and implemented complete Save/Load system for complex game states
  • Created 12 cinematic sequences using Unreal Sequencer
  • Developed all puzzle mechanics and interactive gameplay systems
  • Optimized game performance achieving 60 FPS on target mobile devices
  • Reduced app size by 40% through optimization and asset management
  • Implemented mobile-specific controls and UI systems

Achievements:

  • Selected for 9 Eylül University Game Development Program
  • Multiple game development competition awards
  • Successfully released on iOS and Android platforms
